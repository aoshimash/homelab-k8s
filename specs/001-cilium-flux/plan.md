## Summary

Migrate Cilium from manual Helm installation to GitOps management with Flux. The desired end state is a pinned, PR-driven `HelmRelease` that reconciles the current Cilium configuration without downtime and supports fast rollback via Git revert.

## Technical Context

**Language/Version**: N/A (YAML manifests; Kubernetes CRDs)
**Primary Dependencies**: Flux CD (source/kustomize/helm controllers), Kustomize, Helm chart (Cilium)
**Storage**: N/A
**Testing**: `kubectl` + Flux reconciliation status checks (optionally `cilium status`)
**Target Platform**: Talos Linux Kubernetes cluster (single-node at present; Talos disables default CNI; kube-proxy disabled)
**Project Type**: GitOps infrastructure repository
**Performance Goals**: Zero observed networking downtime/packet loss during migration; fast recovery/rollback via reconciliation
**Constraints**: No persistent imperative `helm install`; pinned version/config; changes via PR only
**Scale/Scope**: One homelab cluster; Cilium in `kube-system`

## Phase 2: Implementation Plan (high level)

### Work items

- Add Flux Kustomization(s) to reconcile `k8s/infrastructure/` from the existing sync root `k8s/flux/`.
- Add Cilium infrastructure manifests:
  - `HelmRepository` for the Cilium chart source.
  - `HelmRelease` for Cilium in `kube-system`, pinned to the currently running version/config.
- Migration procedure (no downtime):
  - Capture current Helm release details (version + values) from the running cluster.
  - Encode the captured values into the Flux `HelmRelease`.
  - Reconcile and verify that the first Flux-managed reconciliation results in no disruptive change.
- Update documentation (`docs/talos-bootstrap.md`) to remove manual `helm install` and describe the Flux-managed path + verification commands.

### Validation plan (acceptance-oriented)

- Flux shows Ready for the Cilium `HelmRepository` and `HelmRelease`.
- Cilium pods remain Ready and nodes remain Ready during cutover.
- Rollback is validated by reverting a Git change and reconciling back to the previous state.
